How to Play Sudoku

Sudoku is a logic-based puzzle game. Fill the 9x9 grid so that each row, column, and 3x3 section contains all numbers from 1 to 9 without repeating.

  • Select a difficulty level and click "New Game" and the timer will start autonatically.
  • The timer can be paused with the "Stop Time" button. "Start Time" can be pressed when you continue the game.
  • Click a cell, then click a number to fill it in.
  • If the cell could contain mutiple numbers add further numbers to try They will appear in red.
  • Click the clear button to delete entries in the cell as needed.
  • Use the toolbar to Undo, Reset, Save, Load, Solve or Print the game.
  • Undo will undo the last ertries in reverse order of entry.
  • Reset will allow you to start the game from the begining.
  • Saving the game will save to the local memory of the web page and pause the timer.
  • Loading the game will retrive any game saved locally including the paused time which will again resume the timer.
  • Solve will solve the game.
  • Print will print a physical copy of your current puzzle to your local printer if you have one available.
